GPS Deviation

Perpendicular distance from each route to the reference route (marked REF — the first selected route with recorded time, if any). "Show" colors that route's map line green → red, from on-course up to its own worst 5% of points — the scale is per-route, not a fixed distance.

Distance Drift

Cumulative distance difference vs. the reference route (the first selected route, marked REF), sampled at equal elapsed time. A steadily diverging value means a device reads consistently long or short.
